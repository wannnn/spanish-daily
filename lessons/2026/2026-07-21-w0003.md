## 基本資訊

- **單字：** estar
- **中文意思：**
  1. 是、處於（表示狀態、情況等暫時或可變的性質）
  2. 在（表示位置、所在地）
  3. 正在（與現在分詞連用，表示進行中的動作）
- **詞性：** 動詞（不規則動詞）

## 用法

- `estar` 是西班牙文兩個「be 動詞」之一（另一個是 `ser`），用來表示**狀態、位置、情況**等相對暫時、可能改變的性質，例如心情、健康狀況、位置、進行中的動作等。
- 用於表達**位置或所在地**，例如 `El libro está en la mesa`（書在桌上）。
- 與現在分詞（`-ando`/`-iendo`）連用構成**現在進行式**，表示動作正在發生，例如 `Estoy comiendo`（我正在吃飯）。
- 常見搭配：
  - `estar + 地點`（`Estoy en casa.` 我在家。）
  - `estar + 形容詞（狀態）`（`Está cansado.` 他累了。）
  - `estar + gerundio（進行式）`（`Está lloviendo.` 正在下雨。）
  - `estar de + 名詞`（`Estoy de vacaciones.` 我在度假。）
  - `estar bien / mal`（`¿Cómo estás?` 你好嗎？——`Estoy bien.` 我很好。）
- 常見固定表達：
  - `estar de acuerdo`（同意）
  - `estar a punto de + 原形動詞`（正要、即將）
  - `estar por + 原形動詞`（尚待、還沒去做某事）
- 容易混淆的詞：
  - `estar` vs `ser`：兩者都譯為「是」，但 `estar` 表示狀態、位置、暫時性的情況（`Ella está cansada.` 她累了——暫時狀態），`ser` 表示本質、身分、不易改變的特徵（`Ella es inteligente.` 她很聰明——本質特質）。部分形容詞用 `ser` 和 `estar` 意思會改變，例如 `ser aburrido`（無聊的，本質）vs `estar aburrido`（感到無聊，暫時狀態）。
  - `estar` vs `haber`：`haber`（`hay`）用於表示某物「存在」，不指明確位置；`estar` 用於指出已知、確定事物的**位置**，例如 `Hay un libro en la mesa`（桌上有一本書，引入新資訊）vs `El libro está en la mesa`（那本書在桌上，指已知的書）。
- 學習者常見錯誤：
  - 混用 `ser` 和 `estar` 描述狀態，例如把「他很累」誤說成 `Él es cansado`（誤將暫時狀態當本質特徵），正確應為 `Él está cansado`。
  - 用 `estar` 表達「存在」，例如誤把「這裡有一隻貓」說成 `Está un gato aquí`，正確應為 `Hay un gato aquí`。

## 詞形變化

`estar` 是**不規則動詞**：現在式第一人稱單數為 `estoy`（而非規則的 `esto`），且除虛擬式外，現在式重音落在字尾（`estás`、`está`），需加重音符號；簡單過去式（`estuve` 系列）為不規則詞根變化，與規則的 -ar 動詞完全不同，需整體記憶。

### Presente（現在式）

| 人稱 | 變化 |
|---|---|
| yo | estoy |
| tú | estás |
| él/ella/usted | está |
| nosotros/as | estamos |
| vosotros/as | estáis |
| ellos/ellas/ustedes | están |

### Pretérito indefinido（簡單過去式）

| 人稱 | 變化 |
|---|---|
| yo | estuve |
| tú | estuviste |
| él/ella/usted | estuvo |
| nosotros/as | estuvimos |
| vosotros/as | estuvisteis |
| ellos/ellas/ustedes | estuvieron |

說明：簡單過去式的詞根改為不規則的 `estuv-`，與規則 -ar 動詞的詞尾變化模式相同，但詞根不規則，需單獨記憶。

### Imperfecto（未完成過去式）

| 人稱 | 變化 |
|---|---|
| yo | estaba |
| tú | estabas |
| él/ella/usted | estaba |
| nosotros/as | estábamos |
| vosotros/as | estabais |
| ellos/ellas/ustedes | estaban |

說明：未完成過去式完全規則，依 -ar 動詞的一般變化規律。

### Perfecto（現在完成式，haber + estado）

| 人稱 | 變化 |
|---|---|
| yo | he estado |
| tú | has estado |
| él/ella/usted | ha estado |
| nosotros/as | hemos estado |
| vosotros/as | habéis estado |
| ellos/ellas/ustedes | han estado |

### Futuro（未來式）

| 人稱 | 變化 |
|---|---|
| yo | estaré |
| tú | estarás |
| él/ella/usted | estará |
| nosotros/as | estaremos |
| vosotros/as | estaréis |
| ellos/ellas/ustedes | estarán |

### Condicional（條件式）

| 人稱 | 變化 |
|---|---|
| yo | estaría |
| tú | estarías |
| él/ella/usted | estaría |
| nosotros/as | estaríamos |
| vosotros/as | estaríais |
| ellos/ellas/ustedes | estarían |

### Subjuntivo Presente（虛擬式現在式）

| 人稱 | 變化 |
|---|---|
| yo | esté |
| tú | estés |
| él/ella/usted | esté |
| nosotros/as | estemos |
| vosotros/as | estéis |
| ellos/ellas/ustedes | estén |

說明：虛擬式現在式詞尾加重音符號（`esté`、`estés`、`estén`），以與規則變化區分讀音重音位置。

### Imperativo（命令式）

| 人稱 | 變化 |
|---|---|
| tú | está |
| usted | esté |
| nosotros/as | estemos |
| vosotros/as | estad |
| ustedes | estén |

## 例句

- **Presente：** *Estoy en la oficina ahora mismo.*
  我現在在辦公室。
  說明：用 `estar` 表示當下所在的位置。

- **Pretérito indefinido：** *Ayer estuvimos en la playa todo el día.*
  昨天我們整天都在海邊。
  說明：描述過去某個明確時間點內、已結束的暫時狀態或位置。

- **Imperfecto：** *Cuando llegué, ella estaba muy nerviosa.*
  我到的時候，她很緊張。
  說明：描述過去某時刻正持續存在的狀態，用未完成過去式。

- **Perfecto：** *Hoy he estado muy ocupado con el trabajo.*
  我今天工作一直很忙。
  說明：現在完成式表示狀態延續到現在、與現在仍有關聯的一段時間。

- **Futuro：** *Mañana estaré en Madrid por trabajo.*
  明天我會因工作在馬德里。
  說明：`estar` 表示未來某時間所處的位置或狀態。

- **Condicional：** *Estaría más tranquilo si terminara el proyecto hoy.*
  如果今天能完成專案，我會比較安心。
  說明：條件式用於表達假設情況下會有的狀態。

- **Subjuntivo：** *Espero que estés bien.*
  希望你一切都好。
  說明：`esperar que` 之後接虛擬式，表達說話者的願望。

- **Imperativo：** *Está tranquilo, todo va a salir bien.*
  放輕鬆，一切都會沒事的。
  說明：對 `tú` 的肯定命令式，用於安撫或勸告對方保持某種狀態。

## 延伸學習

- **同義詞：** `hallarse`、`encontrarse`（均可表示「處於（某狀態）、位於（某處）」，語氣較正式或文學化，可在部分語境中替代 `estar`）。
- **反義詞：** 無明確的反義詞；語意上的對照概念是 `ser`（表示本質、不易改變的特徵，而非暫時狀態）。
