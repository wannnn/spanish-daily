## 基本資訊

- **單字：** hacer
- **中文意思：**
  1. 做、製作（動手完成某件事或物品）
  2. 從事（某活動、運動等）
  3. （天氣）表示……，如 `hace frío`（天氣冷）、`hace sol`（出太陽）
  4. `hace` + 時間，表示「……之前」或「已經……」
- **詞性：** 動詞（不規則動詞）

## 用法

- `hacer` 是西班牙語中使用頻率最高的動詞之一，核心意思是「做」或「製作」，用途非常廣泛，幾乎可以搭配任何具體或抽象的動作受詞，例如 `hacer la tarea`（做作業）、`hacer un pastel`（做蛋糕）。
- 描述天氣時使用第三人稱單數的無人稱句型：`hace frío / calor / sol / viento / buen tiempo`，主詞永遠是隱含的「天氣」，動詞恆為第三人稱單數形式。
- `hace` + 一段時間可表示「……之前」（如 `hace dos años` 兩年前）或搭配現在式表示「已經持續……」（如 `hace dos años que vivo aquí` 我已經住在這裡兩年了）。
- 反身形式 `hacerse` 表示「變成」，例如 `hacerse médico`（成為醫生）、`hacerse tarde`（變晚、來不及）。
- 常見搭配：
  - `hacer la cama / las maletas`（鋪床／收拾行李）
  - `hacer ejercicio / deporte`（做運動）
  - `hacer un viaje / una pregunta`（去旅行／提問）
  - `hacer caso`（理會、聽從）
  - `hacer falta`（需要、必要）
- 常見固定表達：
  - `hacerse cargo (de algo)`（負責、接手處理某事）
  - `hacer de`（扮演……的角色、充當）
  - `no hace falta`（不需要、沒必要）
- 容易混淆的詞：
  - `hacer frío/calor`（天氣冷／熱，無人稱句）vs `tener frío/calor`（人感到冷／熱）：學習者常誤用，例如把「我很冷」說成 `hago frío`，正確應為 `tengo frío`。
  - `hacer` vs `practicar`：兩者都能表示「從事（運動）」，但 `practicar` 更強調「練習、鑽研某項技能」，`hacer` 較泛用、口語。
- 學習者常見錯誤：
  - 混淆天氣句與人的感受句，如誤把「天氣很冷」說成 `Tengo frío hoy`（意思會變成「我今天很冷」）。
  - 未來式與條件式誤用規則詞尾，寫成 `haceré`、`hacería`，正確應為不規則的 `haré`、`haría`。

## 詞形變化

`hacer` 是高度不規則的動詞，不規則之處集中在現在式第一人稱、簡單過去式詞幹、過去分詞，以及未來式／條件式詞幹。

### Presente（現在式）

| 人稱 | 變化 |
|---|---|
| yo | hago |
| tú | haces |
| él/ella/usted | hace |
| nosotros/as | hacemos |
| vosotros/as | hacéis |
| ellos/ellas/ustedes | hacen |

### Pretérito indefinido（簡單過去式）

| 人稱 | 變化 |
|---|---|
| yo | hice |
| tú | hiciste |
| él/ella/usted | hizo |
| nosotros/as | hicimos |
| vosotros/as | hicisteis |
| ellos/ellas/ustedes | hicieron |

### Imperfecto（未完成過去式）

| 人稱 | 變化 |
|---|---|
| yo | hacía |
| tú | hacías |
| él/ella/usted | hacía |
| nosotros/as | hacíamos |
| vosotros/as | hacíais |
| ellos/ellas/ustedes | hacían |

### Perfecto（現在完成式，haber + hecho）

| 人稱 | 變化 |
|---|---|
| yo | he hecho |
| tú | has hecho |
| él/ella/usted | ha hecho |
| nosotros/as | hemos hecho |
| vosotros/as | habéis hecho |
| ellos/ellas/ustedes | han hecho |

### Futuro（未來式）

| 人稱 | 變化 |
|---|---|
| yo | haré |
| tú | harás |
| él/ella/usted | hará |
| nosotros/as | haremos |
| vosotros/as | haréis |
| ellos/ellas/ustedes | harán |

### Condicional（條件式）

| 人稱 | 變化 |
|---|---|
| yo | haría |
| tú | harías |
| él/ella/usted | haría |
| nosotros/as | haríamos |
| vosotros/as | haríais |
| ellos/ellas/ustedes | harían |

### Subjuntivo Presente（虛擬式現在式）

| 人稱 | 變化 |
|---|---|
| yo | haga |
| tú | hagas |
| él/ella/usted | haga |
| nosotros/as | hagamos |
| vosotros/as | hagáis |
| ellos/ellas/ustedes | hagan |

### Imperativo（命令式）

| 人稱 | 變化 |
|---|---|
| tú | haz |
| usted | haga |
| nosotros/as | hagamos |
| vosotros/as | haced |
| ustedes | hagan |

**不規則說明：**

- **現在式：** 只有 `yo` 不規則（`hago`），屬於「-go 動詞」一類（如 `tener → tengo`、`poner → pongo`），其餘人稱規則變化。
- **簡單過去式：** 使用不規則詞幹 `hic-`（強變化過去式），第三人稱單數為 `hizo`——`c` 在 `o` 前改為 `z` 是拼寫規則（保持 /θ/ 或 /s/ 的發音），並非額外的不規則變化。
- **過去分詞：** 不規則形式 `hecho`，而非規則的 `hacido`，`perfecto` 系列時態都以此為基礎。
- **未來式／條件式：** 詞幹縮短為 `har-`（省略 `ce`），並非規則的 `hacer` + 字尾。
- **虛擬式現在式：** 以不規則的現在式 `yo` 詞幹 `hag-` 為基礎規則變化。
- **命令式：** 對 `tú` 的肯定命令式為不規則形式 `haz`，其餘人稱借用虛擬式現在式的形式。

## 例句

- **Presente：** *Yo hago la tarea todos los días después de cenar.*
  我每天吃完晚餐後都會做作業。
  說明：`hacer` 表示「做」某件具體的事，第一人稱單數為不規則形式 `hago`。

- **Pretérito indefinido：** *Ayer hice un pastel de chocolate para mi hermana.*
  昨天我為妹妹做了一個巧克力蛋糕。
  說明：簡單過去式使用不規則詞幹 `hic-`，描述昨天一次已完成的具體動作。

- **Imperfecto：** *De niño, hacía mucho ruido cuando jugaba en casa.*
  我小時候在家玩耍時常常製造很多噪音。
  說明：未完成過去式表示過去反覆或習慣性的動作，此處變化規則。

- **Perfecto：** *Todavía no he hecho las maletas para el viaje de mañana.*
  我還沒為明天的旅行整理行李。
  說明：現在完成式使用不規則過去分詞 `hecho`，表示動作對現在仍有關聯。

- **Futuro：** *Mañana haré ejercicio antes de ir al trabajo.*
  我明天上班前會做運動。
  說明：未來式詞幹縮短為 `har-`，並非規則的 `hacer` + 未來式字尾。

- **Condicional：** *Si tuviera más tiempo libre, haría un curso de cocina.*
  如果我有更多空閒時間，我會去上一門烹飪課。
  說明：條件式同樣使用不規則詞幹 `har-`，搭配虛擬式未完成過去式表示與現實相反的假設情境。

- **Subjuntivo：** *Espero que hagas bien el examen la próxima semana.*
  希望你下週考試考得好。
  說明：虛擬式現在式以不規則的現在式 `yo` 詞幹 `hag-` 為基礎變化，用於表達期望。

- **Imperativo：** *¡Haz la cama antes de salir de casa!*
  出門前先把床鋪好！
  說明：對 `tú` 的肯定命令式使用不規則形式 `haz`，而非規則詞尾。

## 延伸學習

- **同義詞：** `realizar`（較正式，完成、執行某事）、`elaborar`（製作，常用於較精細或複雜之物）、`practicar`（在「從事、練習」某活動的意義上部分重疊）
- **反義詞：** `deshacer`（拆除、撤銷已完成的事）；就「製作／完成」的核心意義而言沒有單一固定的反義詞，需視語境而定。
