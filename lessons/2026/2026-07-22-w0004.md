## 基本資訊

- **單字：** tener
- **中文意思：**
  1. 有、擁有（表示所有、持有）
  2. 表示年齡、感受、狀態（如 `tener frío`、`tener razón`）
  3. 必須（`tener que + 原形動詞`，表示義務）
- **詞性：** 動詞（不規則動詞）

## 用法

- `tener` 是西班牙文使用頻率最高的動詞之一，基本意思是「擁有」，例如 `Tengo un coche`（我有一輛車）。
- 大量用於表達**年齡**，例如 `Tengo veinte años`（我二十歲），字面是「我擁有二十年」。
- 與許多名詞搭配，表達**感受或身體狀態**，中文常譯為「覺得」或「感到」，例如 `tener hambre`（餓）、`tener sed`（渴）、`tener sueño`（睏）、`tener miedo`（害怕）。
- `tener que + 原形動詞` 表示**義務、必須做某事**，例如 `Tengo que estudiar`（我必須讀書）。
- `tener` 加名詞也可表示**擁有的抽象概念**，例如 `tener razón`（有理、說得對）、`tener suerte`（幸運）、`tener prisa`（趕時間）。
- 常見搭配：
  - `tener + 年齡`（`Ella tiene treinta años.` 她三十歲。）
  - `tener + 感覺/狀態名詞`（`Tenemos calor.` 我們覺得熱。）
  - `tener que + 原形動詞`（`Tienes que descansar.` 你必須休息。）
  - `tener ganas de + 原形動詞`（`Tengo ganas de viajar.` 我很想去旅行。）
- 常見固定表達：
  - `tener en cuenta`（考慮到、將…納入考量）
  - `tener lugar`（發生、舉行，用於事件）
  - `no tener idea`（不知道、毫無頭緒）
- 容易混淆的詞：
  - `tener` vs `haber`：`haber` 作助動詞構成完成式（`he tenido` 我曾經有過），也可用 `hay` 表示「存在」；`tener` 則是實義動詞，表示某人「擁有」某物，兩者不可互換，例如不能說 `Hay un coche yo`，而應說 `Tengo un coche`。
  - `tener que` vs `deber`：兩者都可譯為「必須」，但 `tener que` 語氣較強、較口語化，強調外在的必要性；`deber` 語氣較正式，也常表示道德上的義務或推測（`Debe de estar cansado.` 他應該很累。）。
- 學習者常見錯誤：
  - 用 `ser` 或 `estar` 表達年齡或感受，例如誤把「我很餓」說成 `Soy hambre` 或 `Estoy hambre`，正確應為 `Tengo hambre`。
  - 忘記 `tener que` 中的 `que`，誤說成 `Tengo estudiar`，正確應為 `Tengo que estudiar`。

## 詞形變化

`tener` 是**不規則動詞**：現在式除 `nosotros/vosotros` 外詞根母音 `e` 變為 `ie`（`tengo` 除外，屬於 g 化不規則），簡單過去式詞根改為不規則的 `tuv-`，未來式與條件式詞幹縮短為 `tendr-`。

### Presente（現在式）

| 人稱 | 變化 |
|---|---|
| yo | tengo |
| tú | tienes |
| él/ella/usted | tiene |
| nosotros/as | tenemos |
| vosotros/as | tenéis |
| ellos/ellas/ustedes | tienen |

說明：`yo` 形式為特殊的 `g` 化不規則（`tengo`，而非 `*teno`）；其餘人稱除 `nosotros/vosotros` 外，詞根母音 `e` 變為 `ie`（詞幹重音變化，即所謂的鞋形變化動詞）。

### Pretérito indefinido（簡單過去式）

| 人稱 | 變化 |
|---|---|
| yo | tuve |
| tú | tuviste |
| él/ella/usted | tuvo |
| nosotros/as | tuvimos |
| vosotros/as | tuvisteis |
| ellos/ellas/ustedes | tuvieron |

說明：詞根改為不規則的 `tuv-`，詞尾則與其他不規則簡單過去式動詞（如 `estar`、`andar`）共用同一套詞尾模式，需整體記憶。

### Imperfecto（未完成過去式）

| 人稱 | 變化 |
|---|---|
| yo | tenía |
| tú | tenías |
| él/ella/usted | tenía |
| nosotros/as | teníamos |
| vosotros/as | teníais |
| ellos/ellas/ustedes | tenían |

說明：未完成過去式完全規則，依 -er 動詞的一般變化規律。

### Perfecto（現在完成式，haber + tenido）

| 人稱 | 變化 |
|---|---|
| yo | he tenido |
| tú | has tenido |
| él/ella/usted | ha tenido |
| nosotros/as | hemos tenido |
| vosotros/as | habéis tenido |
| ellos/ellas/ustedes | han tenido |

### Futuro（未來式）

| 人稱 | 變化 |
|---|---|
| yo | tendré |
| tú | tendrás |
| él/ella/usted | tendrá |
| nosotros/as | tendremos |
| vosotros/as | tendréis |
| ellos/ellas/ustedes | tendrán |

說明：未來式詞幹不規則地縮短為 `tendr-`（省略母音 `e`），再加上規則的未來式詞尾。

### Condicional（條件式）

| 人稱 | 變化 |
|---|---|
| yo | tendría |
| tú | tendrías |
| él/ella/usted | tendría |
| nosotros/as | tendríamos |
| vosotros/as | tendríais |
| ellos/ellas/ustedes | tendrían |

說明：條件式與未來式共用同一個不規則詞幹 `tendr-`。

### Subjuntivo Presente（虛擬式現在式）

| 人稱 | 變化 |
|---|---|
| yo | tenga |
| tú | tengas |
| él/ella/usted | tenga |
| nosotros/as | tengamos |
| vosotros/as | tengáis |
| ellos/ellas/ustedes | tengan |

說明：虛擬式現在式以現在式 `yo` 形式的不規則詞根 `teng-` 為基礎，加上 -er 動詞虛擬式的標準詞尾。

### Imperativo（命令式）

| 人稱 | 變化 |
|---|---|
| tú | ten |
| usted | tenga |
| nosotros/as | tengamos |
| vosotros/as | tened |
| ustedes | tengan |

說明：`tú` 的肯定命令式為特殊的縮短不規則形式 `ten`（而非依規則變化的 `*tienes` 詞根）；其餘人稱與虛擬式現在式相同。

## 例句

- **Presente：** *Tengo dos hermanos y una hermana.*
  我有兩個哥哥（或弟弟）和一個姊姊（或妹妹）。
  說明：`tener` 表示家庭成員的擁有關係，是最基本的用法。

- **Pretérito indefinido：** *Ayer tuvimos una reunión muy larga.*
  昨天我們開了一場很長的會議。
  說明：簡單過去式描述過去某個已結束的具體事件。

- **Imperfecto：** *De niño, tenía mucho miedo a la oscuridad.*
  我小時候很怕黑。
  說明：未完成過去式描述過去持續存在的習慣性狀態或感受。

- **Perfecto：** *Este mes he tenido mucho trabajo.*
  這個月我一直很忙（工作很多）。
  說明：現在完成式表示狀態延續到現在、與現在仍有關聯的一段時間。

- **Futuro：** *El próximo año tendré más tiempo libre.*
  明年我會有更多空閒時間。
  說明：未來式表示對未來狀態的預測。

- **Condicional：** *Con más dinero, tendría mi propia casa.*
  如果有更多錢，我就會有自己的房子。
  說明：條件式用於表達假設情況下會有的結果。

- **Subjuntivo：** *Ojalá tengas suerte en el examen.*
  希望你考試順利（好運）。
  說明：`ojalá` 之後接虛擬式，表達說話者的祝願。

- **Imperativo：** *Ten cuidado al cruzar la calle.*
  過馬路時要小心。
  說明：對 `tú` 的肯定命令式，用於提醒或叮嚀對方。

## 延伸學習

- **同義詞：** `poseer`（擁有，語氣較正式，常用於書面或法律語境，如財產、資產）。
- **反義詞：** `carecer (de)`（缺乏、沒有），例如 `carecer de dinero`（缺錢）與 `tener dinero`（有錢）相對。
