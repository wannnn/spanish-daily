## 基本資訊

- **單字：** ir
- **中文意思：**
  1. 去、前往（表示移動到某地）
  2. 進行中、進展（描述事情的狀態，如 `ir bien`）
  3. `ir a + 原形動詞`：即將要做某事（近未來式）
- **詞性：** 動詞（不規則動詞）

## 用法

- `ir` 是西班牙文最基本、也是最不規則的動詞之一，核心意思是「去」，例如 `Voy al mercado`（我要去市場）。
- 常與方向介系詞 `a` 搭配，表示前往某地：`ir a + 地點`，例如 `Vamos a la playa`（我們要去海灘）。
- `ir a + 原形動詞` 是西班牙文最常用的**近未來式**結構，用來表示即將發生或計畫中的動作，例如 `Voy a estudiar esta noche`（我今晚要讀書）。
- 可用來描述事情的**進展或狀態**，例如 `¿Cómo te va?`（你過得怎樣？）、`Todo va bien`（一切順利）。
- 反身形式 `irse` 意思是「離開」，與 `ir`（去某處）不同，例如 `Me voy ahora`（我現在要走了）。
- 常見搭配：
  - `ir a + 地點`（`Voy a casa.` 我要回家。）
  - `ir a + 原形動詞`（`Va a llover.` 快要下雨了。）
  - `ir de + 名詞`（`Vamos de compras.` 我們去購物。）
  - `ir en + 交通工具`（`Voy en autobús.` 我搭公車去。）
- 常見固定表達：
  - `¡Vamos!`（走吧！、加油！）
  - `ir de vacaciones`（去度假）
  - `ir al grano`（直接切入重點）
- 容易混淆的詞：
  - `ir` vs `venir`：`ir` 表示「去」（遠離說話者所在地），`venir` 表示「來」（朝說話者所在地移動），例如 `Voy a tu casa`（我要去你家）而對方會說 `Ven a mi casa`（來我家）。
  - `ir` vs `irse`：`ir` 單純描述「去某處」，強調目的地；`irse` 是反身用法，強調「離開此地」的動作本身，例如 `Voy al trabajo`（我去上班）vs `Me voy`（我要走了，不指明去哪）。
- 學習者常見錯誤：
  - 誤以為 `ir a + 原形動詞` 只能用於未來，忽略它在口語中比 `futuro` 更常見、更自然。
  - 混淆 `voy`、`vas`、`va` 等不規則現在式變化，誤套用規則動詞詞尾（如誤寫成 `*io`）。

## 詞形變化

`ir` 是西班牙文中最不規則的動詞之一：現在式、簡單過去式、未完成過去式、虛擬式現在式的詞根都與原形 `ir` 幾乎無關，需整體記憶；未完成過去式甚至是全西語中唯二完全不規則的未完成過去式動詞之一（另一個是 `ser`）。

### Presente（現在式）

| 人稱 | 變化 |
|---|---|
| yo | voy |
| tú | vas |
| él/ella/usted | va |
| nosotros/as | vamos |
| vosotros/as | vais |
| ellos/ellas/ustedes | van |

說明：現在式詞根完全不規則，源自古西語的 `vad-`（與拉丁文 `vadere` 有關），與原形 `ir` 的字面形式無明顯關聯，須整體記憶。

### Pretérito indefinido（簡單過去式）

| 人稱 | 變化 |
|---|---|
| yo | fui |
| tú | fuiste |
| él/ella/usted | fue |
| nosotros/as | fuimos |
| vosotros/as | fuisteis |
| ellos/ellas/ustedes | fueron |

說明：`ir` 的簡單過去式與 `ser`（是）完全相同，兩者在此時態中共用同一套變化形式，須依上下文判斷意思（例如 `Fui a Madrid` 是「我去了馬德里」，`Fui profesor` 是「我曾是老師」）。

### Imperfecto（未完成過去式）

| 人稱 | 變化 |
|---|---|
| yo | iba |
| tú | ibas |
| él/ella/usted | iba |
| nosotros/as | íbamos |
| vosotros/as | ibais |
| ellos/ellas/ustedes | iban |

說明：未完成過去式的詞根 `ib-` 也不規則，是西班牙文中僅有的三個未完成過去式不規則動詞之一（另兩個是 `ser` 的 `era` 與 `ver` 的 `veía`）。

### Perfecto（現在完成式，haber + ido）

| 人稱 | 變化 |
|---|---|
| yo | he ido |
| tú | has ido |
| él/ella/usted | ha ido |
| nosotros/as | hemos ido |
| vosotros/as | habéis ido |
| ellos/ellas/ustedes | han ido |

說明：過去分詞 `ido` 為規則形式（-ir 動詞的標準過去分詞），僅需搭配助動詞 `haber` 的變化。

### Futuro（未來式）

| 人稱 | 變化 |
|---|---|
| yo | iré |
| tú | irás |
| él/ella/usted | irá |
| nosotros/as | iremos |
| vosotros/as | iréis |
| ellos/ellas/ustedes | irán |

說明：未來式規則地以原形 `ir` 為詞幹，直接加上標準未來式詞尾，是少數幾個時態中 `ir` 表現規則的一個。

### Condicional（條件式）

| 人稱 | 變化 |
|---|---|
| yo | iría |
| tú | irías |
| él/ella/usted | iría |
| nosotros/as | iríamos |
| vosotros/as | iríais |
| ellos/ellas/ustedes | irían |

說明：條件式同樣以原形 `ir` 為詞幹，加上規則的條件式詞尾，與未來式的規則性一致。

### Subjuntivo Presente（虛擬式現在式）

| 人稱 | 變化 |
|---|---|
| yo | vaya |
| tú | vayas |
| él/ella/usted | vaya |
| nosotros/as | vayamos |
| vosotros/as | vayáis |
| ellos/ellas/ustedes | vayan |

說明：虛擬式現在式詞根改為不規則的 `vay-`，與現在式的 `voy` 系列詞根不同，是另一套需獨立記憶的不規則形式。

### Imperativo（命令式）

| 人稱 | 變化 |
|---|---|
| tú | ve |
| usted | vaya |
| nosotros/as | vamos |
| vosotros/as | id |
| ustedes | vayan |

說明：`tú` 的肯定命令式為特殊的縮短形式 `ve`（與動詞 `ver` 的 `ve` 同形，須依上下文區分）；`nosotros/as` 的命令式習慣使用現在式的 `vamos`，而非虛擬式的 `*vayamos`。

## 例句

- **Presente：** *Todos los días voy al trabajo en metro.*
  我每天搭捷運去上班。
  說明：現在式表示日常、規律性的動作。

- **Pretérito indefinido：** *El verano pasado fuimos a Barcelona.*
  去年夏天我們去了巴塞隆納。
  說明：簡單過去式描述過去已完成的具體一次性事件。

- **Imperfecto：** *De pequeño, iba al colegio a pie.*
  我小時候都走路去學校。
  說明：未完成過去式描述過去反覆發生的習慣性動作。

- **Perfecto：** *Todavía no he ido a ese restaurante nuevo.*
  我還沒去過那家新餐廳。
  說明：現在完成式表示到目前為止尚未發生、與現在有關聯的經驗。

- **Futuro：** *Mañana iré al médico.*
  我明天要去看醫生。
  說明：未來式表示對未來計畫或安排的直接陳述。

- **Condicional：** *Si tuviera más dinero, iría de viaje a Japón.*
  如果我有更多錢，我就會去日本旅行。
  說明：條件式用於表達假設情況下會發生的結果。

- **Subjuntivo：** *Espero que vayas a la fiesta este sábado.*
  希望你這週六會去派對。
  說明：`esperar que` 之後接虛擬式，表達說話者的期望。

- **Imperativo：** *¡Ve a lavarte las manos antes de comer!*
  吃飯前去洗手！
  說明：對 `tú` 的肯定命令式，用於直接下達指令。

## 延伸學習

- **同義詞：** `dirigirse (a)`（前往、朝……方向去，語氣較正式），例如 `Se dirigió a la estación`（他朝車站走去）。
- **反義詞：** `venir`（來），例如 `venir a casa`（來家裡）與 `ir a casa`（去家裡）方向相反。
